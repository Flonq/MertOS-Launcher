# MertOS Launcher

MertOS Launcher, kişisel projelerime, geliştirme araçlarıma ve önemli bağlantılarıma tek ekrandan ulaşmak için hazırlanmış modern bir geliştirici başlangıç panelidir.

## Özellikler

- Proje kartları
- Geliştirici araçları
- Arama özelliği
- Kategori filtreleme
- Lokal SVG/PNG ikon desteği
- Dark / neon / glassmorphism arayüz
- Responsive tasarım
- HTML, CSS ve JavaScript ile geliştirilmiş hafif yapı

## Kategoriler

- Web
- Game
- Backend
- Araçlar

## Teknolojiler

- HTML
- CSS
- JavaScript
